Aplikacja backendowa stanowiąca kataglog magazynowy. Uwzględnia strukturę podziału na kategorie. 
Zapewnia dwa api do obsługi produktów oraz samych kategorii.

Użyte technologie:
- Django Rest Framework
- Docker
- MongoDB

Wybrałem DRF ponieważ spełnia wszelkie standardy bezpieczeństwa oraz zapewnia sama w sobie założenia architektury REST.

Wymagania:
- Docker 
- Docker Compose

Jak zainstalować:
- pobierz aplikację poprzez git clone lub pobranie paczki i rozpakownie,
- uruchom terminal w folderze parts_warehouse,
- uruchom aplikację porzez komendy: 
    - docker compose build,
    - docker compose up.
